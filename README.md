# kujiecompiler
A compiler for the Kujie toy programming language

# How to run
1. Create a virtual environment `.venv/Scipts/Activate.ps1`
2. Install llvmlite. Run `pip install -r requirements.txt`
3. Run `python main.py`. Runs one of the example files in `/examples/`

# Run unit tests
`python -m unittest discover -s tests`
